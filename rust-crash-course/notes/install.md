# Install

[Rust](https://www.rust-lang.org/)

```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

rustup --version
rustc --version
cargo --version

# update
rustup update

# Build new Cargo project named hello_rust
cargo init hello_rust

# Build project
cargo build

# Format code
cargo fmt

# Run main.rs
cargo run

# Run test
cargo test
```
